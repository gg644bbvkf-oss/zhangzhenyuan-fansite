# 张真源粉丝站 - 图片添加说明

## 📁 图片目录结构

```
images/
├── hero-bg.jpg          # 首页背景图（必填）
├── stage/               # 舞台照片 (1.jpg - 10.jpg)
├── variety/             # 综艺照片 (1.jpg - 10.jpg)
├── formal/              # 正装写真 (1.jpg - 10.jpg)
├── casual/              # 日常随拍 (1.jpg - 10.jpg)
├── concert/             # 演唱会照片 (1.jpg - 10.jpg)
└── news/                # 动态配图 (1.jpg - 6.jpg)
```

## 🖼️ 图片要求

| 目录 | 数量 | 建议尺寸 | 用途 |
|------|------|----------|------|
| hero-bg.jpg | 1张 | 1920x1080 | 首页半透明背景 |
| stage/ | 10张 | 800x1000 | 舞台表演照片 |
| variety/ | 10张 | 800x1000 | 综艺节目照片 |
| formal/ | 10张 | 800x1000 | 正装写真照片 |
| casual/ | 10张 | 800x1000 | 日常照片 |
| concert/ | 10张 | 800x1000 | 演唱会照片 |
| news/ | 6张 | 800x450 | 动态配图 |

## 📥 图片获取方式

1. **百度贴吧**: 搜索"张真源壁纸"找到帖子，里面有百度网盘/一刻相册链接
2. **微博**: 关注 @时代少年团-张真源 保存高清图
3. **抖音**: 保存粉丝分享的饭拍视频截图

## ⚠️ 注意事项

- 图片命名必须按顺序: 1.jpg, 2.jpg, ...
- 如果某个分类不足10张，只放已有的即可
- 图片加载失败时会显示占位符
- 添加新图片后刷新页面即可看到
